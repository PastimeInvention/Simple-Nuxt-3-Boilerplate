# Simple Nuxt 3 Boilerplate


## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

Create Env file

```bash
cp env.example .env

```

## Development Server

Start the development server on `http://localhost:3000`

```bash
yarn run dev
```

## Production

Build the application for production:

```bash
yarn run build
```

Locally preview production build:

```bash
yarn run preview
```
